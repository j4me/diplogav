Программный продукт  NGN (version 1.0)

представляет собой электронное учебное пособие по курсу сети связи

и состоит из следующих разделов:

 

 

Содержание. Дает возможность  студенту выбрать  интересующий раздел.

 

Тест. Тест (от англ. - испытание, исследование) - стандартизированное задание, по результатам выполнения которого судят о  знаниях, умениях и навыках испытуемого. В контексте данной программы тест - это ряд вопросов, предложенных тестируемому, на каждый из которых даётся несколько вариантов ответа. Заметим, что среди данных вариантов ответа обязательно должен присутствовать верный. Задача тестируемого выбрать вариант (или варианты) ответа, который, с его точки зрения является правильным. Тестов в данном учебном пособии несколько: после некоторых разделов содержания для проверки остаточных знаний, и итоговый тест, по результатам которого можно оценить работу студента с учебником.

 

Результаты. После ответа на все предложенные в тестах вопросы, имеется возможность просмотреть результаты тестирования. Результаты тестирования представлены в отдельном окне. По итоговым результатам выставляется оценка за продемонстрированные знания.

 

Справка. Раздел Справка включает в себя: словарь, список сокращений используемых в учебнике, список литературы по теме, рекомендации ITU-T .

 

Выход. После завершения работы с программой, при нажатии кнопки с надписью "Выход", осуществляется, как ни странно, выход из программы.
